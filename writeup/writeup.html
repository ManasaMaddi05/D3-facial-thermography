4. Technical Implementation

  • Stack: D3.js v7, plain HTML/CSS, no frameworks
  • Data Loading: `d3.csv()` + merging FLIR/ICI, adding `group` column
  • Data Join: keyed by region name for smooth `.join()` transitions
  • Update Logic: `updateFace(region, temp)` recolors selected + neighbor regions
  • Performance: in‑memory caching; minimal DOM ops on slider drift
  • Deployment: hosted on GitHub Pages (master branch /docs folder)
